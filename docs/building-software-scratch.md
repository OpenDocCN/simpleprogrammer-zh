# 从头开始构建软件

> 原文：<https://simpleprogrammer.com/building-software-scratch/>

很多软件开发人员喜欢把事情复杂化。他们得到的可能是简单容易的东西，由于大多数人在互联网上说的话，使它变得复杂。

软件要简单。也就是说，很容易理解为什么这么多开发人员很难从零开始构建软件。在他们看来，这是一件非常复杂和困难的事情，他们会完全陷入其中。

所以，问题是…你如何从零开始构建软件？
