在心流中，每一个行动，每一个决定，都从上一个无缝地产生。在这种状态下，我们如此专注于手头的任务，以至于其他一切都被抛在脑后。行动和意识融合在一起。我们的自我意识消失了。我们的时间观念扭曲了。而且，正如研究表明的那样，性能的各个方面都突飞猛进。
