# 成功的程序员心态:不要偷懒！

> 原文：<https://simpleprogrammer.com/successful-programmer-mindset/>

你觉得你无法继续你的生活。你觉得被困住了，你做的任何事情似乎都不起作用。你被困在自己的生活中。

你有工作、学校、家庭和朋友要处理。你看不到出路…我该怎么办，约翰？

嗯，有很多因素在这场灾难旋风中起了作用……其中之一就是懒惰。

你有没有停下来问自己是否在偷懒？如果没有，是时候这么做了！
